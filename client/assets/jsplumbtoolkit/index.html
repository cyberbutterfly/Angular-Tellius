Welcome to the Toolkit edition of jsPlumb. If you're not already doing this, the best thing to do is to host this entire directory structure on a webserver somewhere.

Getting Started

If you have Grunt installed you can get started straight away with a copy of an existing demo, or with an almost empty app you can use as a template.

Cloning an existing demo

grunt clone --app=database-visualizer --o=<your output directory>
Valid values for `-app` are:
  • database-visualiser
  • flowchart-builder
  • absolute
  • circular
  • hierarchy
  • paths
  • hello-world
  • imageWidget
  • multiple
  • skeleton
  • spring

Creating a new skeleton app

grunt create --o=<your output directory>
